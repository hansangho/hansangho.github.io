String

  • substr : 자를 문자열의 첫번째 위치, 자를 문자열의 갯수

Number

  • toFixed : 소숫점 이하 자리 반올림
  • toPrecision : 지정한 갯수의 숫자로 자름

        var aa = 123.45678;

        aa.toFixed(3); // "123.457"

        aa.toPrecision(3); // "123"
        aa.toPrecision(4); // "123.5"
        aa.toPrecision(10); // "123.4567800"

이스케이프 처리

  • encodeURI() / decodeURI()
  • encodeURIComponent() / decodeURIComponent()

        var urlString = 'http://finance.naver.com/news/news_read.nhn?article_id=0006999901&office_id=003';
         
        encodeURI( urlString ); // "http://finance.naver.com/news/news_read.nhn?article_id=0006999901&office_id=003"
        encodeURIComponent( urlString ); // "http%3A%2F%2Ffinance.naver.com%2Fnews%2Fnews_read.nhn%3Farticle_id%3D0006999901%26office_id%3D003"
